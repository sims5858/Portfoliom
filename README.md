# Sezer Portfolio

Sezer için hazırlanmış tek sayfalık kişisel CV ve portfolyo web sitesi.

## İçerik

- E-ticaret operasyon uzmanlığı
- Entegrasyon yönetimi
- SEO ve içerik çalışmaları
- Referans projeler
- Platform ve araç bilgileri
- WhatsApp ve iletişim alanları

## Dosyalar

- `index.html`
- `styles.css`
- `script.js`
- `server.js`

## Yerelde Çalıştırma

Node.js yüklüyse proje klasöründe:

```bash
node server.js
```

Ardından:

```text
http://127.0.0.1:3000
```

## Yayınlama

Site statik yapıda hazırlandığı için Vercel gibi platformlara kolayca deploy edilebilir.
